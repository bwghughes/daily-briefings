## 🔥 Top Stories

### Google Ships Gemma 4: Open-Weight Models with Native Audio + Vision
Google DeepMind released **Gemma 4** — four Apache 2.0 licensed models (2B, 4B, 31B, 26B-MoE) purpose-built for advanced reasoning and agentic workflows. This is Google's first open-weight release with **native audio input** for speech recognition and understanding alongside vision capabilities.

The smaller E2B and E4B models use "Per-Layer Embeddings (PLE)" for extreme parameter efficiency on mobile/edge devices. Google emphasizes "intelligence-per-parameter" — part of the industry-wide race to make small models genuinely useful.

Already available in LM Studio, Ollama, and Google AI Studio. Android AICore Developer Preview integrates Gemma 4 directly into the OS.

**Sources:** [Google Blog](https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/), [Ars Technica](https://arstechnica.com/ai/2026/04/google-announces-gemma-4-open-ai-models-switches-to-apache-2-0-license/), [Android Developers Blog](https://android-developers.googleblog.com/2026/04/AI-Core-Developer-Preview.html), [Mashable](https://mashable.com/article/google-releases-gemma-4-open-ai-model-now-open-source-how-to-try-it)

### Anthropic's Opus 4.6 Dominates Agentic Coding Benchmarks

Platform note: Anthropic is retiring the 1M token context beta for Sonnet 4.5/4 on April 30, 2026.

**Sources:** [Anthropic](https://www.anthropic.com/news), [Wikipedia](https://en.wikipedia.org/wiki/Claude_(language_model)), [Anthropic Docs](https://platform.claude.com/docs/en/about-claude/models/whats-new-claude-4-6)

OpenClaw (launched Nov 2025 as Clawdbot) has become the **fastest-growing open source project in GitHub history** with 346,000 stars as of early April. However, security researchers have flagged **135,000+ publicly exposed instances** with risky configurations.

The New Stack published a comparison: "[OpenClaw vs. Hermes Agent: The race to build AI assistants that never forget](https://thenewstack.io/persistent-ai-agents-compared/)."

## 💻 Apple Silicon & Local AI

### Google AI Edge Gallery App Ships for iPhone
Google released the **Google AI Edge Gallery** app for iPhone — the first official vendor app for trying local models on iOS. Ships with Gemma 4 E2B (2.54GB download) and E4B, plus Gemma 3 models. Features include image Q&A, audio transcription (up to 30s), and a "skills" demo showcasing tool calling across 8 interactive widgets.

Fast, genuinely useful, but conversations are ephemeral (no permanent logs).

**Source:** [Simon Willison](https://simonwillison.net/2026/Apr/6/google-ai-edge-gallery/)

### Apple ML Research: Simple Self-Distillation for Code Generation
Apple researchers published "[Embarrassingly Simple Self-Distillation Improves Code Generation](https://arxiv.org/abs/2604.01193)" on arXiv (April 2026). The paper shows **LLMs can dramatically improve code generation** by sampling their own outputs at certain temperatures and fine-tuning on them — no external data required.

Tested on Llama 3.1 (8B) and Qwen3 (4B/30B) across instruct and thinking modes. Implications for on-device model improvement via self-supervised learning.

**Sources:** [arXiv](https://arxiv.org/abs/2604.01193), [Neuronad](https://neuronad.com/ai-news/apples-embarrassingly-simple-secret-to-smarter-ai/)

### Android AICore Developer Preview with Gemma 4
Google announced the **AICore Developer Preview** for Android, integrating Gemma 4 directly into the OS. Planned updates include tool calling, structured output, system prompts, and thinking mode in Prompt API.

**Source:** [Android Developers Blog](https://android-developers.googleblog.com/2026/04/AI-Core-Developer-Preview.html)
