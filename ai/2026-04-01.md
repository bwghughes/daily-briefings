## 🚨 Security & Incidents

**Axios npm Supply Chain Attack**
- Axios package (101M weekly downloads) compromised with RAT malware in v1.14.1 and 0.30.4 via leaked npm token
- Malicious dependency "plain-crypto-js" installed credential stealers
- Attack linked to recent Trivy security scanner exploit
- [Socket Security](https://socket.dev/blog/axios-npm-package-compromised)

**LiteLLM Malware Attack Affected 47,000 Downloads**
- PyPI packages 1.82.7 and 1.82.8 compromised for 46 minutes
- 2,337 dependent packages, 88% with unpinned versions
- Credential stealer harvested SSH keys, AWS credentials, Docker config, crypto wallets
- [FutureSearch](https://futuresearch.ai/blog/litellm-hack-were-you-one-of-the-47000/)

## 🍎 Apple Silicon & Local AI

### Major Performance Breakthrough

**Ollama Adopts MLX Framework**
- Ollama 0.19 preview now built on Apple's MLX framework
- **Performance gains on M5 chips:**
  - Prefill: 1.6x faster (57% improvement)
  - Decode: 1.93x faster (93% improvement)
- Currently supports Qwen3.5-35B-A3B model (requires 32GB+ unified memory)
- Includes NVFP4 quantization support for production parity
- Works with OpenClaw, Claude Code, Codex coding agents
- [Ollama Blog](https://ollama.com/blog/mlx) | [Ars Technica](https://arstechnica.com/apple/2026/03/running-local-models-on-macs-gets-faster-with-ollamas-mlx-support/) | [MacRumors](https://www.macrumors.com/2026/03/31/ollama-now-runs-faster-apple-silicon-macs/) | [Hacker News Discussion](https://news.ycombinator.com/item?id=47582482)
