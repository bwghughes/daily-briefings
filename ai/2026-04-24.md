### Apple Three-Layer Inference Architecture
BEP Research breaks down why Apple's architecture is unique: **Layer 1** (on-device, 2B-7B via MLX + Foundation Models), **Layer 2** (Private Cloud Compute on M5 chips — same silicon DNA as consumer devices), **Layer 3** (Gemini partner for frontier overflow). No other company has this — Google has TPU/Tensor split, Microsoft has NVIDIA/Qualcomm split, Amazon has Trainium/nothing-on-device split. Same LPDDR interface, same MLX, same memory architecture from iPhone to PCC server node. [BEP Research](https://bepresearch.substack.com/p/the-other-memory-wall-part-2)

### M5 MacBook Pro + Mac Studio
M5 Pro/Max launched March 2026 — first chiplet-based consumer silicon. 18-core CPU shared across Pro/Max, 4× faster LLM prompt processing via Neural Accelerator in each GPU core. M5 Ultra inevitable (two Max dies fused) → 80-core GPU, up to 256GB unified memory, potentially ~1.2TB/s bandwidth. [Apple Newsroom](https://www.apple.com/newsroom/2026/03/03/apple-introduces-m5-pro-and-m5-max/)

### Qwen3.5-397B on M3 Ultra: 38 tok/s
Ivan Fioravanti benchmarked Qwen3.5-397B-A17B (4-bit, MoE, 17B active params) on M3 Ultra/512GB at 38 tok/s short context, 24 tok/s at 128k context. This is on M3 Ultra — two generations old.
